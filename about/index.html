About this site

I use tools and services that make my life much simpler.

This site is built on Jekyll, an static site generator and hosted on Github Pages under the github.com/ignaciodenuevo.github.io repository.

The site is written in Sass, and built from scratch.

The fonts used are Guardian Bold Egyptian for headings, Brando Sans and Semibold for body text and Input for code.

The photographs of the home page and my portrait were taken by Lea Farren

The icons used are Feather Icons.
