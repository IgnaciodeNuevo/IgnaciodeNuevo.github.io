Curso CSS

¿Otro curso de CSS más?

La idea este curso es compartir MI experiencia y prácticas a la hora de escribir bien CSS desde el punto de vista práctico, así como casos de uso para algunas situaciones.

He empezado a crear un curso de CSS (en español) para sacarlo en 2018. 😊

Será gratuito y sin necesidad de registro.

¿Algo que os gustaría que incluyera? 🤔 (A parte de flexbox, grid etc)

— Ignacio Villanueva (@IgnaciodeNuevo) 15 de noviembre de 2017

¿Qué es y qué no es este curso?

No es un curso desde cero.

No pretendo crear cátedra ni ser dogmático.

¿Qué veremos en el curso?

Estos son algunos (no todos) los apartados que veremos.

  • Especificidad y cascads
  • Valores
  • Unidades de medida (em, rem, porcentajes, fr...)
  • Colores
  • Tipografía Web
  • Uso práctico de "Flexbox"
  • Uso práctico de Grid Layout
  • Filtros y "Blend-Modes"
  • Animaciones
  • Custom Properties ("Variables CSS")
  • Selectores que deberías conocer (+, *, >)
  • Arquitecturas: 7-1 e ITCSS
  • Linting
  • Media-Queries
  • Buenas prácticas
  • Como nombrar clases (BEM)
  • Refactorizar CSS

¿Cuándo estará el curso?

Saldrá de manera completamenta gratuita a lo largo de 2018. Eso implica que podría salir el 31 de diciembre. ¡Paciencia!
