Curso CSS

¿Otro curso de CSS más?

Este curso no es otro más, no veremos como usar el último framework (Bootstrap, Bulma...) o herramienta. Hablaré sobre procesos y las bases de la maquetación.

La idea este curso es compartir MI experiencia y prácticas a la hora de escribir bien CSS desde el punto de vista práctico, así como casos de uso.

¿Qué es y qué no es este curso?

No es un curso desde cero.

El curso no pretende crear cátedra ni ser dogmático.

¿Qué veremos en el curso?

Estos son algunos (no todos) los apartados que veremos.

  • Especificidad y cascada
  • Unidades de medida (em, rem, porcentajes, fr...)
  • Selectores que deberías conocer (+, *, >)
  • Colores
  • Tipografía Web
  • Media-Queries
  • Uso práctico de "Flexbox"
  • Uso práctico de Grid Layout
  • Filtros y "Blend-Modes"
  • Animaciones
  • Custom Properties
  • Buenas prácticas
  • Convenciones de nombrado
  • Arquitecturas: 7-1 e ITCSS
  • Linting
  • Refactorizar CSS

¿Cuándo estará el curso?

Saldrá de manera completamenta gratuita a lo largo de 2018. Eso implica que podría salir el 31 de diciembre. ¡Paciencia!
